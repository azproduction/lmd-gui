Welcome to LMD builder!

Big JavaScript application cause huge startup latency. A 1Mb of JavaScript initializes about ~600-3000ms! without touching any part of DOM. It evals module only when they are required. LMD is inspired by AMD and provides similar module interface

There is some important actions

Info – to see detailed build info: build plugins, detailed modules information.

Rebuild – LMD will create package using current build config. Additional build info will be provided.

Watch – LMD will watch all your files declared in current build config. LMD will rebuild all project if one of them changes. This is a background process.

Server – to start stats server. Server provides detailed package analytics: code coverage, usage, load time, dynamic depends. This is a background process.
